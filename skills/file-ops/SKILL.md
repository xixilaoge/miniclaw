# File Operations 技能

提供文件系统操作的最佳实践指导。

## 使用

帮助用户高效、安全地进行文件操作。

### 支持的操作

| 操作 | 说明 |
|------|------|
| 查找文件 | 按名称、类型、大小查找 |
| 搜索内容 | 在文件中搜索文本模式 |
| 批量重命名 | 安全的批量文件重命名 |
| 目录遍历 | 递归处理目录结构 |

### 示例

- "找出所有大于 10MB 的文件"
- "在 src 目录搜索包含 'TODO' 的文件"
- "批量重命名图片文件"

## 最佳实践

1. **安全第一** - 删除操作前先确认
2. **使用版本控制** - 重要修改前先提交
3. **备份重要数据** - 不可恢复的操作要谨慎
