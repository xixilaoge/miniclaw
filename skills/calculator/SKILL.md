# Calculator 技能

一个简单的计算器技能，可以执行基本数学运算。

## 使用

用户可以用自然语言描述计算需求，技能会解析并计算结果。

### 示例

- "1 + 1 等于几？" → 2
- "10 乘以 5" → 50
- "100 除以 4" → 25
- "50 减去 20" → 30

## 支持的运算

- 加法：加、加上、+、plus
- 减法：减、减去、-、minus
- 乘法：乘、乘以、×、*、times
- 除法：除、除以、÷、/、divided by
